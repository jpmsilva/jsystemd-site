About jsystemd Parent

Aggregator project for all jsystemd modules

Project Modules

This project has declared the following modules:

Name Description
jsystemd The core library for systemd integration. This component does not depend on Spring, and can be used in any Java application.
jsystemd starter module for Spring Boot Spring boot starter that provides auto-configuration for jsystemd, effectively turning any spring boot application into a proper system service under systemd.
